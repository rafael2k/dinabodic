# _DinaBodic

DinaBodic is a 2D starship platform game written in Java.

For more information, read the report about the game:

![](screen1.png)

# Running

Download the "DinaBodic.zip" file and inside the "DinaBodic" folder, run:

```
java -jar DinaBodic.jar
```
